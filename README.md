# Chat-rede

## Rodando o projeto:

Necessário baixar as dependências do projeto com: **npm install**

Rodar o servidor:
**node app/server.js**

Para cada conexão de usuário é necessário abrir outro terminal e rodar...:
**node app/client.js**

## Utilização
Após entrar no chat digite seu nome. Para sair, digite "exit".
